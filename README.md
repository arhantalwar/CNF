## QUICK START

Dependencies

- [raylib](https://www.raylib.com/)

```console
$ gcc main.c -o main -lraylib
$ ./main 7 (For still images)
$ ./main 7 -f (For animated images)
```
You can take screenshot of the image you like by pressing the 'S' key while the image is shown.
